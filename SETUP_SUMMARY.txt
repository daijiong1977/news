                      QUICK START (5 STEPS)
================================================================================

Step 1️⃣: Get API Token
   Go to: https://buttondown.email/settings/api
   Create token if needed
   Copy the token

Step 2️⃣: Setup Token
   bash setup_buttondown_token.sh
   Paste token when prompted
   → Creates .env file

Step 3️⃣: Create Test Subscribers
   python3 buttondown_subscriber_manager.py
   → Creates 4 subscribers with tags
   → Asks to delete others (you can skip)
   → Verifies setup

Step 4️⃣: Prepare Campaigns
   python3 buttondown_send_campaigns.py
   → Creates 4 email drafts
   → Prints sending instructions

Step 5️⃣: Send Emails (Manual via Dashboard)
   Go to: https://buttondown.email/dashboard
   For each campaign:
   - Find draft by subject
   - Click Send
   - Filter by tag (Level: Easy, etc.)
   - Send!

Result: Each subscriber gets their level-specific email!

================================================================================
                      TROUBLESHOOTING
================================================================================

Problem: "BUTTONDOWN_API_TOKEN not set"
Solution:
  export BUTTONDOWN_API_TOKEN='your_token_here'
  python3 buttondown_subscriber_manager.py

Problem: "Invalid API token"
Solution:
  Get new token: https://buttondown.email/settings/api
  bash setup_buttondown_token.sh

Problem: Emails not sending
Solution:
  1. Verify subscribers exist: python3 buttondown_subscriber_manager.py
  2. Verify tags are correct in dashboard
  3. Verify campaign tag filter matches

Problem: Images not showing
Solution:
  1. Check image URLs are accessible
  2. Update domain from localhost:8000 to production
