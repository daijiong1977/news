## 🚀 3-Step Quick Start

```bash
# 1. Install dependency
pip install requests

# 2. Initialize database  
python3 deepseek_processor.py --init

# 3. Process articles
python3 deepseek_processor.py --batch-size 5
```

Done! Results stored in database.
