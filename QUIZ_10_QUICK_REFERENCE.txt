# Quiz System Quick Reference - 10 Questions with 3 Word Types

## 📊 New Question Structure

**10 Questions Per Article** organized into **3 word types**:

| Word Type | Count | Difficulty | Question Style | Example |
|-----------|-------|-----------|----------------|---------| 
| **What** | 3-4 | ⭐ Easy | "What is...?" | "What does innovation mean?" |
| **How** | 3-4 | ⭐⭐ Medium | "How does...?" | "How does the technology work?" |
| **Why** | 3-4 | ⭐⭐⭐ Hard | "Why is...?" | "Why is this important?" |

---

## 🎮 CLI Commands

```bash
# Get all 10 questions for article 16
python3 deepseek_processor.py --quiz 16

# Get one question by ID
python3 deepseek_processor.py --quiz-id 42

# Get all questions (all articles)
python3 deepseek_processor.py --all-quiz

# Get only "what" questions (easy)
python3 deepseek_processor.py --quiz-type what_questions

# Get only "how" questions (medium)
python3 deepseek_processor.py --quiz-type how_questions

# Get only "why" questions (hard)
python3 deepseek_processor.py --quiz-type why_questions
```

## 🎲 Random Quiz Example

```python
import json
import random
import subprocess

# Get all questions
result = subprocess.run(
    ["python3", "deepseek_processor.py", "--all-quiz"],
    capture_output=True, text=True
)
questions = json.loads(result.stdout)

# Pick random 5 questions
quiz = random.sample(questions, k=5)

for q in quiz:
    print(f"[{q['word_type'].upper()}] {q['question']}")
    for i, opt in enumerate(q['options']):
        print(f"  {chr(65+i)}) {opt}")
```

## 🎯 Difficulty-Based Game

```python
def get_easy_questions(count=3):
    result = subprocess.run(
        ["python3", "deepseek_processor.py", "--quiz-type", "what_questions"],
        capture_output=True, text=True
    )
    qs = json.loads(result.stdout)
    return random.sample(qs, k=count)

def get_hard_questions(count=3):
    result = subprocess.run(
        ["python3", "deepseek_processor.py", "--quiz-type", "why_questions"],
        capture_output=True, text=True
    )
    qs = json.loads(result.stdout)
    return random.sample(qs, k=count)
```

## 🎲 Random Selection Ideas

**3-Question Daily Quiz**
```python
daily = random.sample(all_questions, 3)
```

**Balanced Quiz (1 of each type)**
```python
by_type = {}
for q in all_questions:
    wtype = q['word_type']
    if wtype not in by_type:
        by_type[wtype] = []
    by_type[wtype].append(q)

balanced = [
    random.choice(by_type['what']),
    random.choice(by_type['how']),
    random.choice(by_type['why'])
]
```

easy = get_by_type('what_questions')[:3]
